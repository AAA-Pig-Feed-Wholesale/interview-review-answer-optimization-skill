# 面试复盘与回答优化

## Skill 简介

这是一套面向技术岗的面试复盘与回答优化 skill。  
它的目标不是“帮你包装经历”，而是基于你的真实材料，完成三件事：

- 还原面试中出现了什么问题
- 判断你原回答哪里有问题
- 给出更稳、更贴岗、更能落到口语表达中的答法

它适合在面试结束后使用，也适合在已有模拟面试记录时使用。

## 第一版适用岗位

第一版主要面向技术岗，重点支持：

- AI 应用开发
- RAG
- Agent
- 后端

如果岗位仍属于技术岗，但不完全在上述范围内，也可以使用；只是岗位特化判断会相对保守。

## 输入材料说明

### 必需输入

至少建议提供以下两类材料：

- 简历
- 面试对话记录

这是第一版的最低完整输入配置。

### 可选输入

可进一步补充：

- 岗位 JD
- 项目材料 / 项目说明 / 项目总结
- 自我介绍稿
- 其他背景材料

### 输入完整度对效果的影响

如果只有简历和面试记录，这套 skill 仍可正常完成：

- 候选人画像简短版
- 问题提取
- 题型判断
- 原回答问题诊断
- 回答优化
- 整场短板总结

如果补充岗位 JD，会增强：

- 岗位贴合分析
- 考点判断
- “这题为什么问” 的识别准确度

如果补充项目材料，会增强：

- 个性化回答质量
- 项目题 / 场景题 / 系统设计题的细节支撑
- 回答中的真实素材调用能力

如果缺少简历，则只能做通用复盘，不做强个性化回答映射。

## 输出模式简介

这套 skill 支持 3 种输出模式：

### `extract_only`

只做抽取和整理，适合先确认：

- 面试官到底问了什么
- 题目属于什么类型
- 当前材料是否足够

它也会保留问题簇与原始追问链，方便后续继续深入。

### `review`

以复盘和诊断为主，适合先看：

- 原回答的问题在哪里
- 每题在考什么
- 整场面试的主要短板是什么

这是默认模式。

### `optimize`

以逐题回答优化为主，适合直接获得：

- 更好的回答思路
- 个性化回答
- 优秀回答
- 记忆锚点
- 30 秒 / 90 秒答法
- 高价值题的追问训练标签

对于高价值题，这个模式会额外增强“现场可用性”，而不是只把回答写长。

## 这套 skill 的核心特点

### 1. 候选人画像构建

这套 skill 不会一上来就逐题改答案。  
它会先根据简历、项目材料、JD 等信息，构建一个简短的候选人画像，用来确定：

- 你的目标岗位方向
- 最相关项目经历
- 后续优先调用哪些真实素材
- 简历里的高风险项
- 更适合你的优势表达方向
- 回答风格建议

这一步的作用是让后续优化更贴近你的真实背景，而不是套模板。

### 2. 双层回答机制 + 现场训练增强

这套 skill 的回答优化不是只给一种答案，而是区分两层：

- 个性化回答：基于你的真实经历来写，不虚构
- 优秀回答：按理想候选人标准给出更成熟的参考答法

在当前版本中，高价值题还会按需要增加：

- 记忆锚点 / Golden Phrase
- 30 秒极简版
- 90 秒面试展开版
- 原始追问链
- 追问耐受度 / 当前成熟度

当前版本不只是“复盘工具”，也更像“面试训练稿生成器”：

- 能输出结构化复盘
- 能补记忆锚点
- 能补临场话术
- 能补追问训练
- 能补 30 秒 / 90 秒实战口语版答案
- 能显式做真实性、降级和风险边界控制

这样做的好处是：

- 你能知道“基于我真实情况，最稳妥该怎么答”
- 也能知道“这类题更成熟的标准答案大概长什么样”
- 还能知道“这题现场应该先怎么接、追问下能不能扛住”

对于八股题、概念题、原理题或与个人经历弱相关的问题，通常只输出优秀回答，不强行个性化。

## 快速开始

### 最小使用方式

至少提供：

- 简历
- 面试对话记录

然后直接说明你希望的输出模式，例如：

- “帮我先做 review”
- “帮我按 optimize 模式逐题优化”
- “先 extract_only，把问题提出来”

如果你已经知道自己要练高价值题，也可以要求：

- 先保留原始追问链
- 对重点题补 30 秒版 / 90 秒版
- 对边界题补记忆锚点

### 如果只有简历和面试记录

这套 skill 仍可完成一版完整基础复盘，包括：

- 候选人画像简短版
- 问题提取
- 题型判断
- 原回答问题诊断
- 更好的回答思路
- 通常可做的个性化回答
- 整场短板总结

同时：

- `extract_only` 可以先把问题簇和原始追问链抽出来
- `optimize` 可以对重点题补现场训练增强字段

### 如果补充 JD 和项目材料

效果会更强，尤其体现在：

- 更准确地判断岗位考点
- 更稳定地做岗位贴合分析
- 更充分地调用项目细节
- 个性化回答更具体、更可信
- 更容易产出业务分层案例
- 更容易做具像的证据绑定 / 幻觉治理示例
